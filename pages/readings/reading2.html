Que es GITHUB

GitHub es una plataforma que trabaja como sistema de gestor de proyectos y control de versiones de código, de igual forma esta plataforma se puede catalogar como un repositorio gigante y también como una de red social orientada a desarrolladores, que permite compartir c{odigo y trabajar de forma colaborativa con otras personas en diferentes partes del mundo. Esta plataforma provee herramientas para planificar y asignar tareas de los proyectos y realizar un seguimiento del trabajo del equipo de desarrollo

Para crear una cuenta en este repositorio, se debe acceder a GitHub e ingresar los datos requeridos.
