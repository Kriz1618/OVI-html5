Que es CSS3

CSS "Cascading Style Sheets”" es un lenguaje de diseño gráfico que permite crear y modificar la parte visual de un documento estructurado escrito en un lenguaje de marcado, ha tenido una evolución en el tiempo Y actualmente se encuentra en la versión 3, como indica su actual nombre CSS3. CSS3 es usado pricipalmente para definir la presentación de documentos web e interfaces de usuario escritas en HTML.

Los principales propiedades de CSS3 son:

  • border-radius Permite redondear las esquinas de algunos componentes.
  • box-Shadow Permite aplicar profundidad a tus elementos.
  • text-shadow Similar a box-shadow, pero aplicado al texto.
  • text-outline Permite crear un efecto de borrador en el texto.
  • text-stroke Define un fondo en el texto.
  • text-overflow Se utiliza para cortar el texto que exceda el ancho de su contenedor.
