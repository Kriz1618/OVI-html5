Que es GIT

GIT es uno de los mejores sistemas de control de versiones que existe en la actualidad, fue diseñado por Linus Torvalds, su fin es organizar de forma confiable y eficiente, las versiones de proyectos que poseen una gran cantidad de carpetas y archivos de código fuente. GIT permite generar y restaurar las versiones de los archivos que sean necesarias, lo cual genera bastante confianza puesto que siempre es posible restaurar la versión especifica de un archivo.

Los principales comandos de GIT son:

  • git ini: Se utiliza para iniciar un proyecto dentro de una carpeta especifica.
  • git help: Muestra una lista con los comandos más utilizados en GIT.
  • git pull: Nos permite descargar la información desde un repositorio remoto y un branch especifico.
  • git add: Agrega al repositorio los archivos nuevos o modificados que se indiquen.
  • git commit: Hace commit de los archivos que han sido agregados con el comando git add.
  • git push: Sube los cambios que se encuentran agregados al anterior commit.
