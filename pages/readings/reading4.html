Que es HTML5

HTML5 es la última versión de HTML "HyperText Markup Language", es un lenguje de marcado de texto y no un lenguaje de programación como muchos mencionan, puesto que no cuenta con la estructura de un lenguaje de programación.
HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros

Las principales etiquetas o elementos de HTML5 son:

  • <area> Define un área dentro de un mapa de imagen.
  • <!DOCTYPE> Define el tipo de docuemento.
  • <b> Define texto en negrita.
  • <body> Define el cuerpo del documento.
  • <header> Define la sección de encabezado del docuemnto.
  • <a> Define un hipervínculo.
